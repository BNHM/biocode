Batch Upload Biocode Records
Instructions to batch upload Excel records to Biocode:
  1. Download the Biocode Excel Template (note separate tabs for collecting events, specimen records, and tissues.)
  2. Delete or move any columns as desired EXCEPT for required fields (in yellow).
  3. Fill out the Excel sheet.
  4. Use "Save As" to save the sheet as Text (Tab-delimited *.txt). NOTE: On Mac Save As "Text (Windows)".
  5. Use the appropriate Browse button below to select the file to upload.
  6. Click Submit.
Note Collecting events must be in the database before specimen records can be uploaded. Specimen records must be in the database before tissue records can be uploaded.

Collecting Events Any collector names on this file must already be in the database. Click here to add a new collector.


Specimen Records Each specimen record in this file must have a collecting event number that is already in the database.


Tissue Records Each tissue record must have a specimen number that is already in the database.
